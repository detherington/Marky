Marky 1.0.2

New: Find & Replace

  • Cmd+F opens a unified find bar in all three editing modes.
  • Cmd+Option+F opens with Replace visible.
  • Cmd+G / Cmd+Shift+G navigate matches.
  • Live match count, case-sensitivity toggle, click-to-toggle replace row.
  • Works seamlessly in Markdown, Split, and Preview modes. In Split, matches highlight on both panes simultaneously.
  • Replace All is a single undo step.

New: Cmd+P Quick File Switcher

  • Cmd+P opens a fuzzy-search palette for any file in the workspace folder.
  • Type a few characters — filename matches are boosted over path matches.
  • ↑ / ↓ to navigate, Enter to open, Esc to close.
  • Click the dim backdrop to dismiss.

Under the hood

  • Automatic updates via Sparkle (from 1.0.1) are working — this is the first update delivered through that channel.
  • Fixed a rendering loop that could occur when switching editor modes with the find bar open.
