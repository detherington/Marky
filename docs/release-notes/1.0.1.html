Marky 1.0.1

First public release.

  • Three editing modes: raw Markdown, split view, WYSIWYG preview
  • Sidebar file browser with live file-system watching and sort options
  • Tabbed editing
  • Syntax highlighting for headings, bold, italics, code, links, lists, and blockquotes
  • Keyboard shortcuts for inline formatting (Cmd+B, I, E, K, D)
  • Automatic updates via Sparkle
