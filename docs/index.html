Marky

A lightweight markdown editor for macOS.

Download for macOS

Three editing modes

Raw markdown with syntax highlighting, side-by-side split view, or true WYSIWYG editing directly in the rendered preview.

Sidebar file browser

Open a folder and browse all your markdown files at a glance. Sort by last opened, modified, or name.

Native and fast

Built with SwiftUI and WebKit. Tabs, keyboard shortcuts, signed and notarized by Apple, with automatic updates.
